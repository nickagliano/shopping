# shopping

Personal deal-watcher harness. Maintain a watchlist of items, poll marketplace plugins on an interval, get notified when leads appear.

> **Status: early / rough.** This harness is a 2/10 — underdeveloped and lightly used. The architecture has potential (plugin-based marketplaces, swappable notifiers) but it needs a lot more work before it's reliable or pleasant to use. Treat it as a starting point, not a finished tool.

See [CUSTOMIZE.md](CUSTOMIZE.md) for ports and customization.
